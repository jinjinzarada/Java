컬렉션 : 여러객체(다수의 데이터)를 모아놓은 것
컬렉션 프레임워크 : 컬렉션을 다루기 위한 표준화 된 프로그래밍 방식 (쉽고 편리함)
컬렉션 클래스 : 다수의 데이터를 저장할 수 있는 클래스 ex)ArrayList,HashSet

핵심 인터페이스
1. List : 순서 있고 중복 허용 ex) 대기자 명단
2. Set : 순서 없고 중복 안됨 ex) 집합
3. Map : 키(key)와 값(Value)의 쌍으로 저장
	순서없고 키는 중복 안됨 값은 중복 가능 ex) 아이디와 비밀번호
